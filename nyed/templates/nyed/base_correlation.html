Correlation Analysis

This section performs data analysis to investigate if there is any correlation between students' acadmic performance and certain external factors that may have affected their performace, such as school's average class size, student family's financial status, etc.

To view the result of a Correlation Analysis, please

  1. Select a "Correlation Variable": Average Class Size or Free Lunch Eligibility
  2. Choose a "Year" for which the source data is available
Data Source:
All analyses are conducted using the open datasets provided by New York State Education Department
